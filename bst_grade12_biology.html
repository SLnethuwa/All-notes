☁️ කාලගුණ සාධක — ජීව පද්ධති වලට බලපෑම

කාලගුණය (Weather) යනු කෙටි කාලීන වායුගෝලීය තත්ත්ව වේ. දේශගුණය (Climate) යනු දිගු කාලීන (30+ අවුරුදු) සාමාන්‍ය කාලගුණ රටාවයි.

🌡️ ප්‍රධාන කාලගුණ සාධක ┌──────────────────────────────────────────────────────┐ │ කාලගුණ සාධක │ │ │ │ 🌧️ වර්ෂාපාතය ☀️ සූර්යාලෝකය 🌡️ උෂ්ණත්වය │ │ 💨 සුළං 💧 සාපේක්ෂ ආර්ද්‍රතාව │ │ 🔽 වායුමණ්ඩල පීඩනය │ │ │ │ ↓ ↓ ↓ ජීව පද්ධති වලට බලපෑම ↓ ↓ ↓ │ │ │ │ 🌱 ශාක වර්ධනය 🐄 සත්ත්ව ක්‍රියාකාරිත්වය │ │ 🐟 ජලජ ජීවීන් 🌾 කෘෂිකර්ම නිෂ්පාදනය │ └──────────────────────────────────────────────────────┘

🌧️ වර්ෂාපාතය

ශාක වර්ධනයට, ජල සැපයුමට, වගා සෘතුවට බලපායි

☀️ සූර්යාලෝකය

ප්‍රකාශ සංස්ලේෂණය, ශාක ඵල ගැනීම හා ගෙවසේ ශාන්තතාව

🌡️ උෂ්ණත්වය

බීජ අංකුරය, ක්‍ෂුද්‍රජීවී ක්‍රියාකාරිත්වය, සත්ත්ව ශ්වසනය

💧 ආර්ද්‍රතාව

ශාක ඉෙදළිය, රෝග ව්‍යාප්තිය, ජල ෛවෂ්ය

🔑 ප්‍රධාන කරුණ: AWS (Automated Weather Station) — ස්වයංක්‍රීයව දත්ත රැස් කර ඩිජිටල් ගබඩා කරයි. කාලගුණ පුරෝකථනය (Weather Forecasting) — ගොවිතැනේ කාල සූදානම සඳහා ඉතා වැදගත්.

ප්‍රශ්න හා පිළිතුරු — කාලගුණ සාධක

🌍 පස — ජීව පද්ධති වල පස් ගවේෂණය

පස යනු ශාක, ක්‍ෂුද්‍රජීවීන්, ගල් කාෂ, වාතය සහ ජලය අඩංගු සංකීර්ණ ස්වාභාවික සම්පතකි.
